# OG Images — Image by LUI

All images should be **1200x630px**, JPEG or PNG, under 1MB.
Photographic with subtle LUI branding — not text-heavy graphics.

## Required files:

| File | Page | Notes |
|---|---|---|
| `services-overview.jpg` | Services Overview (EN + ES) | Lu in professional setting with client, brand colors visible |
| `occasion-styling.jpg` | Occasion Styling (EN + ES) | Lu or styled client at event, showing styled pieces |
| `image-consulting.jpg` | Image Consulting (EN + ES) | Lu conducting color analysis with Pantone drapes |
| `faq.jpg` | FAQ (EN + ES) | Branded graphic with LUI logo + 'FAQ' on brand palette |

One image per page (shared between EN and ES versions).
